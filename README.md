# Time Scout PWA

Versão simples instalável do Time Scout.

## Como usar no Android

1. Publique esta pasta em um site HTTPS, por exemplo GitHub Pages.
2. Abra o link no Chrome do celular.
3. Toque em “Instalar Time Scout” ou no menu do Chrome > “Adicionar à tela inicial”.
4. Depois de instalado, ele abre em tela cheia e funciona offline.

## Observação

Abrir o `index.html` direto como arquivo funciona para usar o scout, mas o modo aplicativo/PWA completo precisa estar em um endereço HTTPS.
